AUCTF 2020 - Forensic - Block 2

--------------------------------------------------------------------------------------------------
Prompt:
Billy runs a minecraft server. He wants to know which block was rolled back. Can you help him?
NONSTANDARD FLAG FORMAT: Insert the unix timestamp of the row into auctf{time_stamp}
https://drive.google.com/open?id=1lxgtuF3zthgrCNEwaJ8k_EdoEymukTMG
---------------------------------------------------------------------------------------------------

When you download the file, you'll notice that it's an .sql.
To read it, I upload it to the mySQL service on Ubuntu.
In the description, we notice that it talks about "rolled back", as if by chance, in the database, one of the columns is called "rolled_back", and is either 0 or 1, so we execute the query:
SELECT * FROM block2 WHERE rolled_back=1;
We get a single tuple response, so we get the timestamp: 1581060905.

We get our flag: auctf{1581060905}.
