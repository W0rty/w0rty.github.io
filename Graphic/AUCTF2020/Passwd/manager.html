AUCTF 2020 - Password Cracking - Manager

----------------------------------------------------------------
Manager
There might be a flag inside this file. The password is all digits.
NOTE: The flag is NOT in the standard auctf{} format
----------------------------------------------------------------

When we look at the file extension given with this challenge, we notice it's a keepass file that contains passwords (.kdbx).
So I run keepass, and load the file. As I expected, it asks me for a password to be able to open the database that contains the passwords, in this case, our flag.
Also, in the challenge statement, we are told that the password only contains numbers.
So I run keepass2john, to find the password hash: keepass2john manager.kdbx > manager.hash .

So I retrieve the password hash.
So I use hashcat again to break the password:
hashcat64 -a 3 -m 13400 manager.hash ?d?d?d?d?d?d?d?d?d

We get the password: 157865
So we restart keepass, load the file, enter the password, we have our flag: y0u4r34r34lh4ck3rn0w#!$1678.
